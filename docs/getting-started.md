# Getting Started

## Install

First clone the repo and change directories

```
git clone https://github.com/cgcardona/earth.git
cd earth
```

Build the app

```
cargo build
```

Run the test suite and benchmarks

```
cargo test
cargo bench
```

Run the `earth` command w/ a help flag to see options

```
earth --help
```

Once you are ready to start

```
earth
```
